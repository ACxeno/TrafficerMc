# TrafficerMC

⚠️ This version of the program is no longer actively maintained.

## Features
TrafficerMC includes the following main features:
- Anti AFK
- Chat/command automation
- Inventory control
- Movement control
- Auto reconnect
- Script system
- Proxy support
- Account management
- Name generation

Supports Minecraft versions 1.8.x – 1.21.11

---

## Scripting (Summary)
The scripting system allows the bot to automate actions such as:
- Sending chat messages
- Delaying actions
- Using items
- Managing inventory
- Controlling movement
- Disconnecting / reconnecting

---

## Build Guide (Summary)

**Requirements:** NodeJS

```bash
git clone https://github.com/ACxeno/TrafficerMC
cd TrafficerMC
npm install
````

**Build:**

* Windows: `build:win`
* Mac: `build:mac`
* Linux: `build:linux`

**Run in development mode:**

```bash
npm run dev
```
